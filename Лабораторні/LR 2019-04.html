Робота з операторами циклів та ассоціативними масивами

Мета роботи

Навчитись створювати сценарії на мові РНР для опрацювання асоціативних масивів

Обладнання

Персональний комп'ютер. Пакет програм XAMPP. Текстовий редактор Sublime Text 3 або IDE NetBeans. Web-браузер Chrome, Firefox, Opera

Хід роботи

  1. Впевнитись, що пакет XAMPP встановлено та web-сервер Apache запущений
  2. Перейти до каталогу C:\xampp\htdocs\ та очистити його
  3. Впевнитись, що файл є валідним, використавши валідатор HTML-коду https://validator.w3.org/. За необхідності, виправити помилки та зауваження
  4. В каталогу C:\xampp\htdocs\ створити файли з іменем index.php

  5. Відкрити index.html та змінити його наступним чином:

    <!DOCTYPE html> <html> <head> <title>Test</title> <meta charset="utf-8"> </head> <body> <h1>Тестова сторінка. ЛР3</h1> <p>Текст текст текст текст текст текст текст текст текст текст текст текст текст текст текст текст текст текст текст текст текст текст текст</p> <time> <?php echo date("Y-m-d H:i:s"); ?> </time> </body> </html>

    Зберегти файл та оновити вкладку у браузері

  6. Перейти за адресою http://127.0.0.1/index.php або http://localhost/index.php та впевнитись, що завантажилася ваша HTML-сторінка

  7. Виконати індивідуальне завдання

    1. У змінні РНР помістити ваші прізвище, ім'я, дату народження
    2. Вивести в HTML інформацію зі змінних двома способами: з використанням echo та конструкції <?= ?>
    3. Використовуючи функцію date() вивести дату народження у трьох форматах: ДД.ММ.РРРР, РРРР-ММ-ДД, ДД місяць РРРР
  8. Впевнитись, що всі вихідні HTML-сторінки є валідними, використавши валідатор HTML-коду https://validator.w3.org/. За необхідності, виправити помилки та зауваження

  2. Яким чином можна вивести значення змінної?
  3. Які аргументи можна задати для функції date()?
